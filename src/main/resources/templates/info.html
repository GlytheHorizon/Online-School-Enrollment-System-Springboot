About This Project

Online School Enrollment System — a Spring Boot web application built as a personal experiment by Jerwin E. Cruz.

I have no prior knowledge in Spring Boot or Java web development. This entire project was vibe coded — built through AI assistance, trial and error, and a lot of curiosity.

The application follows an MVC + DAO architecture using Spring Boot 3, Thymeleaf templates, Spring Data JPA, and MySQL. It manages student registration, course offerings, enrollment tracking, and tuition payments.
